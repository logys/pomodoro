1.0 Description

Play is the state when the Pomodoro works.

1.1 Tick Play002

When there is tick the Pomodoro shall increment their counter in a 10 ms.

1.2 Button Play003

When the state changes to pause the Pomodoro shall save their current acummulated time.

Parent links: Button003 Toggling

1.3 Into Play004

When there is a transition from pause the Pomodoro shall continue from the accommulated time.

Parent links: Button003 Toggling

1.4 Finished Play005

If the acummulated time reaches the session time the Pomodoro transit to Poweroff state.
