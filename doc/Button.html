1.0 Push Button

Description of the functionalityes related to the push button.

2.1 WeakeUp Button002

When the Pomodoro is in Poweroff state, if the user push the button then the Pomodoro shall change his state to Play

Attribute Value
Implemented True

2.2 Toggling Button003

When the Pomodoro is on Play/Pause if the user push the button the Pomodoro shall change their state to Pause/Play

Attribute Value
Implemented True

2.3 PowerDown Button004

If the button is pressed for two seconds the Pomodoro shall change ther state to Poweroff.

Attribute Value
Implemented True
